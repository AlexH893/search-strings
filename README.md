# Flowbite-Svelte-Starter

## Feature

- Svelte
- SvelteKit
- Tailwind CSS
- Flowbite
- Flowbite-Svelte
- ESlint
- Typescript
- Playwright
- Prettier
- Svelte-heros(Heroicons)
- Darkmode activated

## Installation

```sh
git clone git@github.com:shinokada/flowbite-svelte-starter.git my-demo
cd my-demo
pnpm i // or npm i
pnpm run dev // or npm run dev
```
